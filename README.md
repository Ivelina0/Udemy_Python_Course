 - Plotly & Cufflinks

 - Data Capstone Projects: 911 calls, finance project
 - Linear reression project
 - Logistic regression
 - K Nearest Neighbors
 - Decision Trees and Random Forests
 - Support Vector Machines
 - K Means Clustering
 - Principal Component Analysis
 - Recommender Systems
 - Natural Language Processing
 - Big Data and Spark
 - Deep Learning
 - Scipy Notes
